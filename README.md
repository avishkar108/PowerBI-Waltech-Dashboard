# Power BI Project – Waltech Sales Dashboard

This project is a Power BI dashboard developed to analyze sales data for a fictional retail company, Waltech.

## 📊 Features

- Sales, Profit, and Quantity Analysis
- Product Category & Sub-Category Insights
- Geo-visualization (State/City-level)
- Customer behavior & Payment Mode Trends
- Interactive filters and drill-downs

## 🛠 Tools Used

- Power BI
- DAX
- Power Query
- CSV (Data Sources)

## 📁 Files

- `Waltech Dashboard.pbix`: Power BI dashboard file
- `Details.csv`, `Orders.csv`: Source data
- `Screenshots/`: Images of the dashboard

## 🔗 How to Use

Open `Waltech Dashboard.pbix` using Power BI Desktop to explore the report.
